# FUTURE_DS_01
Business Sales Data Analysis Dashboard

Description:
This project focuses on analyzing business sales data to uncover valuable insights into revenue trends, product performance, customer purchasing behavior, and regional sales distribution. The dashboard transforms raw transactional data into interactive visualizations, enabling businesses to make informed, data-driven decisions.

Objective:
The primary objectives of this project are to:
-Analyze overall sales and revenue performance
-Identify top-selling and high-revenue products.
-Determine the most profitable product categories.
-Evaluate regional sales performance.
-Discover sales trends over time.
-Present key business KPIs through an interactive dashboard.

Tools & Technologies:
-PowerBI-Interactive dashboard creation and visualization
-Microsoft Excel – Data cleaning, preprocessing, and exploratory analysis
-Dataset: Online Retail Sales Dataset

Key Insights:
-Identified monthly and yearly revenue trends.
-Discovered the highest revenue-generating products.
-Identified the best-performing product categories.
-Compared sales performance across different countries/regions.
-Measured overall business KPIs including:
  .Total Revenue
  .Total Order
  .Total Quantity Sold
  .Average Order Value
